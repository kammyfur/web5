Documentation Minteck Projects - JavaScript non fonctionnel

Introduction

JavaScript est une fonction des navigateurs actuels permettant d'éxecuter des actions du côté du client (donc après le chargement de la page) tels que des animations, musiques, vidéos, le rendu d'une page Web, et bien plus encore.
Lorsque ce composant est désactivé ou indisponible pour une quelconque raison, certaines pages Web peuvent refuser de ce charger.

Dans cette documentation, nous allons voir comment activer JavaScript et vérifier si celui-ci fonctionne correctement

Réparations

JavaScript est-il activé ?

Il se peut que pour des raisons de sécurité, votre navigateur et/ou votre ordinateur est bloqué l'exécution de JavaScript depuis des sites non approuvés.
Minteck Projects n'utilisons pas JavaScript à des fins malveillantes et malintentionnées.

Pour réactiver JavaScript si vous utilisez une extension qui le désactive, dirigez vous vers la documentation de celle-ci...

JavaScript est-il supporté par votre navigateur ?

Afin d'être surs de supporter les navigateurs actuels, certains éditeurs de sites Web ne testent plus certains navigateurs, tels que ceux qui sont trop vieux. JavaScript est une fonctionnalité présente depuis peu de temps...
En outre, il se peut que certains navigateurs ne supportent pas JavaScript, ou encore certaines fonctions dans JavaScript.

Internet Explorer est supporté depuis sa version 11,
Microsoft Edge depuis sa version 16 (ou peut être activé à partir de sa version 15),
Mozilla Firefox (bureau) depuis sa version 60 (ou peut être activé à partir de sa version 54),
Google Chrome (bureau) depuis sa version 61 (ou peut être activé à partir de sa version 60), Safari sur MacOS et sur iOS depuis sa version 11,
Opera depuis sa version 48 (ou peut être activé à partir de sa version 47),
Le navigateur d'Android depuis sa version 67,
Google Chrome (mobile) depuis sa version 70 et
Mozilla Firefox (mobile) depuis sa version 63.
Les navigateurs n'ayant pas été cités ne sont pas supportés par JavaScript
